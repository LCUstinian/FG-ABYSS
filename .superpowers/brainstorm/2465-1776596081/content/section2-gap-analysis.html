🎨 全局样式规则缺口

❌ Mono Font 强制列表未全面落实 (URL、session ID、版本号等)
❌ Focus ring 未设置（所有可聚焦元素缺统一的 box-shadow 高亮）
❌ Disabled 状态 未统一 (opacity: 0.38, cursor: not-allowed)
❌ Cursor 规则 未全面落实（多处交互元素无 cursor: pointer）
❌ Naive UI themeOverrides未完整实现（缺DataTable/Menu/Modal等覆盖）
❌ 路由过渡动画 fade 存在但时长未对齐（规格: enter 120ms / leave 80ms）
❌ Skeleton shimmer 加载占位骨架屏未实现
❌ 亮色主题 多处样式会出现对比度问题（卡片无阴影等）
📋 结论：修复计划范围
在原有「交互骨架修复」基础上，新增「规格对齐 UI 精修」模块。两个部分合并为一个统一的计划执行，按以下顺序：

Layer 0（本次新增）：Naive UI themeOverrides 完整实现 · 全局 CSS 规则落实
Layer 1（基础设施）：Capabilities · 滚动契约 · z-index · 高度变量
Layer 2（布局组件）：Titlebar · Sidebar · StatusBar 精修
Layer 3（各视图）：逐视图对齐规格 · Settings 补全5个面板
