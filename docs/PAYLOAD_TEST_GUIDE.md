   ```bash
   php -S localhost:8000 -t C:\path\to\shell\directory
   ```


## 🐛 可能遇到的问题

### 问题 1：生成按钮点击后无反应

**可能原因**：
- 前端与后端通信失败
- API 绑定未正确生成

**解决方案**：
1. 检查浏览器控制台是否有错误
2. 检查 Go 后端日志
3. 重新生成 Wails 绑定：`wails generate`

### 问题 2：生成的代码为空

**可能原因**：
- 模板渲染失败
- 模板文件不存在

**解决方案**：
1. 检查后端日志中的模板加载信息
2. 验证模板文件路径
3. 检查 `payload_template.go` 中的内置模板

### 问题 3：复制功能不工作

**可能原因**：
- 剪贴板 API 权限问题
- 浏览器安全策略

**解决方案**：
1. 检查浏览器是否允许剪贴板访问
2. 手动选择代码复制
3. 使用下载功能替代

### 问题 4：下载的中文乱码

**可能原因**：
- 文件编码问题

**解决方案**：
1. 使用 UTF-8 编码打开文件
2. 检查生成的代码是否包含 BOM 头
