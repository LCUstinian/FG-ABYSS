② 滚动容器契约（Scroll Contract）

当前问题：多层 overflow:hidden 嵌套，导致内容被裁剪、点击无法穿透。新规则：

❌ 现在（有问题）
.app-shell overflow:hidden
└─ .app-main overflow:hidden
   └─ .content-area overflow:hidden
      └─ .home-view overflow-y:auto ✓
      └─ .payload-view 无滚动 ✗
      └─ .plugin-view  无滚动 ✗
✅ 修复后（统一规则）
.app-shell overflow:hidden 边界
└─ .app-main overflow:hidden 边界
   └─ .content-area overflow:hidden 边界
      └─ 每个 View 自己负责滚动
         overflow-y:auto + height:100%

📌 规则：框架层只做 overflow:hidden 边界，滚动由各视图自己管理。

③ z-index 层级规范

LAYER 0
页面内容
z-index: auto
→
LAYER 1
Sidebar / Titlebar
z-index: 10
→
LAYER 2
浮动面板 / Dropdown
z-index: 100
→
LAYER 3
Toast / Dialog
z-index: 1000+

④ 视图高度基准

/* styles/app-shell.css — 新增 CSS 变量 */
:root {
  --titlebar-h: 48px;
  --statusbar-h: 28px;
  --content-h: calc(100vh - var(--titlebar-h) - var(--statusbar-h));
}

/* 所有顶层视图统一使用 */
.view-root { height: var(--content-h); overflow: hidden; }

各 View 的根元素改用 .view-root 语义类，高度来源统一，不再依赖隐式继承。
