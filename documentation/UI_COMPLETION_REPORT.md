## 🔧 技术特性

### CSS 变量命名规范

```css
/* 主色系 */
--{semantic}-{shade}: {value}
例：--primary-500, --success-600

/* 基础变量 */
--{property}-{level}: {value}
例：--bg-primary, --text-secondary

/* 间距系统 */
--space-{n}: {value}
例：--space-2, --space-4

/* 圆角系统 */
--radius-{size}: {value}
例：--radius-md, --radius-lg

/* 动画系统 */
--{type}-{name}: {value}
例：--duration-fast, --ease-standard
```

### 向后兼容策略

所有旧变量名都通过 CSS 变量映射到新系统：
```css
:root {
  /* 新变量 */
  --primary-500: #3b82f6;
  --bg-primary: #ffffff;
  
  /* 旧变量映射 */
  --active-color: var(--primary-500);
  --bg-color: var(--bg-primary);
}
```

**效果**：现有代码无需任何修改即可享受新设计系统

### 性能优化策略

1. **GPU 加速**
   - 动画使用 transform 和 opacity
   - 避免使用 top/left 实现动画

2. **合理使用 will-change**
   - 仅在关键动画元素上使用
   - 动画结束后移除

3. **CSS 变量**
   - 运行时开销极小
   - 现代浏览器原生支持

4. **避免过度设计**
   - 通配符选择器谨慎使用
   - 避免过深的选择器嵌套

## 🚀 使用指南

### 快速开始

1. **全局样式已自动引入**
   ```typescript
   // main.ts 已引入
   import './styles/global.css'
   ```

2. **使用 CSS 变量**
   ```css
   .my-component {
     background: var(--bg-primary);
     color: var(--text-primary);
     border: 1px solid var(--border-subtle);
   }
   ```

3. **使用按钮组件**
   ```html
   <button class="btn btn-primary btn-lg">主要按钮</button>
   <button class="btn btn-secondary">次要按钮</button>
   <button class="btn btn-ghost">幽灵按钮</button>
   ```

4. **使用卡片组件**
   ```html
   <div class="card">
     <div class="card-header">
       <h3 class="card-title">卡片标题</h3>
     </div>
     <div class="card-content">
       卡片内容
     </div>
   </div>
   ```

5. **使用动画工具类**
   ```html
   <div class="animate-fade-in">淡入效果</div>
   <div class="animate-slide-up">上滑效果</div>
   <div class="animate-scale-in">缩放效果</div>
   ```

### 最佳实践

1. **优先使用 CSS 变量**
   - 保持主题一致性
   - 便于维护和修改

2. **使用工具类快速布局**
   - `.flex`, `.items-center`, `.justify-between`
   - `.gap-2`, `.gap-4`, `.gap-6`

3. **合理使用动画**
   - 重要交互使用 emphasis 缓动
   - 持续动画使用 smooth 缓动
   -  playful 场景使用 bounce 缓动

4. **遵循响应式设计**
   - 使用响应式工具类
   - 测试不同屏幕尺寸
