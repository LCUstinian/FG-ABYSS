# FG-ABYSS 哥斯拉 WebShell 管理工具复刻与增强计划

## 项目愿景

打造一款**功能全面、安全可靠、操作便捷**的下一代 WebShell 管理工具，完整复刻哥斯拉（Godzilla）的全部核心功能，并在安全性、性能和用户体验方面实现全面超越。

---

## 当前状态评估

### ✅ 已完成（30-35%）
- **基础架构**：DDD 架构清晰，分层合理
- **数据库层**：SQLite + GORM，支持软删除、分页、搜索
- **UI/UX**：现代化界面，支持深色模式、响应式设计
- **设置系统**：完整的主题、语言、强调色自定义
- **项目管理**：完整的 CRUD 操作和回收站功能
- **WebShell 管理**：基础的 CRUD 和列表展示

### ❌ 缺失核心功能（65-70%）
- WebShell 连接与控制引擎
- 文件管理模块
- 命令执行模块
- 数据库管理模块
- Payload 生成引擎
- 插件系统

## 技术架构升级建议

### 当前架构
```
┌─────────────────────────────────────┐
│         Frontend (Vue 3)            │
│  - Components                       │
│  - Composables                      │
│  - Utils                            │
└──────────────┬──────────────────────┘
               │
         Wails Runtime
               │
┌──────────────┴──────────────────────┐
│          Backend (Go)               │
│  ┌─────────────────────────────┐    │
│  │  Application Layer          │    │
│  │  - Handlers                 │    │
│  │  - Services                 │    │
│  └─────────────────────────────┘    │
│  ┌─────────────────────────────┐    │
│  │  Domain Layer               │    │
│  │  - Entities                 │    │
│  │  - Repositories (Interface) │    │
│  └─────────────────────────────┘    │
│  ┌─────────────────────────────┐    │
│  │  Infrastructure Layer       │    │
│  │  - Repositories (Impl)      │    │
│  │  - Database (SQLite)        │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
```

### 建议升级
1. **添加连接管理层**：专门管理 WebShell 连接
2. **添加编码器层**：统一处理编解码逻辑
3. **添加插件层**：支持插件扩展
4. **添加缓存层**：提升性能

## 开发规范

### 代码规范
- Go: 遵循 Effective Go 和 Go Code Review Comments
- Vue 3: 遵循 Vue 3 最佳实践和 Composition API 规范
- TypeScript: 严格模式，禁止使用 `any`
- 注释：所有公开 API 必须有文档注释

### 测试规范
- 单元测试覆盖率 > 80%
- 集成测试覆盖核心功能
- E2E 测试关键用户流程

### 安全规范
- 所有用户输入必须验证和清理
- 敏感数据必须加密存储
- 所有 API 调用必须认证
- 定期进行安全审计
