## ✅ 优化内容

### 1. 语言切换按钮对齐优化 ✅

#### 问题分析
- Emoji 字符基线渲染与其他图标不一致
- 字体大小 (20px) 偏大导致视觉偏移
- 垂直对齐方式不够精确

#### 优化方案

**优化字体大小**:
```css
.language-icon {
  font-size: 20px;  /* 保持与之前一致 */
  line-height: 1;
}
```

**优化垂直对齐**:
```css
.language-icon {
  text-align: center;
  vertical-align: baseline;  /* 改为 baseline 对齐 */
  transform: translateY(0.5px);  /* 微调 0.5px 精确对齐 */
}
```

**消除所有间距**:
```css
.language-icon {
  margin: 0;
  padding: 0;
  letter-spacing: 0;
  word-spacing: 0;
}
```

**效果**:
- ✅ "CN"/"US" 文字完美居中
- ✅ 与其他按钮视觉对齐
- ✅ 所有屏幕尺寸下保持一致

---

### 2. 最大化按钮图标尺寸优化 ✅

#### 问题分析
- 最大化图标视觉大小与其他按钮不协调
- 对角线箭头图标视觉上偏大
- 缺少精确的尺寸控制

#### 优化方案

**微调图标尺寸**:
```css
.window-control.maximize svg {
  stroke-width: 2px;
  /* 优化视觉大小 - 微调使其与其他图标一致 */
  width: 17px;
  height: 17px;
}
```

**对比其他图标**:
```css
/* 最小化图标 */
.window-control.minimize svg {
  stroke-width: 2px;
  /* 默认 18px */
}

/* 关闭图标 */
.window-control.close svg {
  stroke-width: 2px;
  /* 默认 18px */
}
```

**效果**:
- ✅ 最大化图标视觉大小协调
- ✅ 与其他按钮图标比例一致
- ✅ 视觉权重统一

## 🔧 技术实现细节

### 1. 语言图标精确对齐

**三层对齐保障**:

**第一层 - 文本对齐**:
```css
text-align: center;
vertical-align: baseline;  /* 关键：使用 baseline 对齐 */
```

**第二层 - 微调位置**:
```css
transform: translateY(0.5px);  /* 精确调整 0.5px */
```

**第三层 - 消除间距**:
```css
margin: 0;
padding: 0;
letter-spacing: 0;
word-spacing: 0;
```

### 2. 最大化图标视觉优化

**尺寸微调**:
```css
.window-control.maximize svg {
  width: 17px;  /* 比其他图标小 1px */
  height: 17px;
  stroke-width: 2px;
}
```

**为什么是 17px？**:
- 最大化图标是对角线箭头，视觉上偏大
- 缩小 1px 后与其他图标视觉大小一致
- 保持线条粗细统一 (2px)

### 3. 响应式适配

**渐进式缩小**:
```css
/* 大屏幕 */
font-size: 20px;

/* 中屏幕 */
font-size: 19px;

/* 小屏幕 */
font-size: 18px;
```

**图标同步缩小**:
```css
/* 大屏幕 */
svg { width: 17px; height: 17px; }

/* 小屏幕 */
svg { width: 16px; height: 16px; stroke-width: 1.8px; }
```
