## 🐛 已知问题

### 问题 1: 界面显示空白 - ✅ 已修复

**现象**: 载荷模块内容区域显示空白，只有一个大卡片和"生成 Payload 预览代码"文字

**原因**:
- n-tabs 和 n-grid 样式设置不当导致内容区域被压缩
- v-model 绑定语法不正确
- 容器高度设置不合理

**解决方案** (2026-03-17 已实施):
1. ✅ 修改 v-model 绑定：`v-model:value` → `v-model`
2. ✅ 优化 n-tabs 样式：添加 `height: 100%` 和 `min-height: 0`
3. ✅ 优化 n-tab-pane 样式：使用 flex 布局
4. ✅ 优化 generator-container：添加 flex 布局支持
5. ✅ 优化 n-grid：添加 `flex: 1` 和 `min-height: 0`
6. ✅ 优化 n-card：添加 `height: 100%` 样式
7. ✅ 优化内置模板卡片：添加 `flex-shrink: 0`

**修改的文件**:
- `frontend/src/components/PayloadsContent.vue` - 模板和样式修复
